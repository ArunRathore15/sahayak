# Sahayak
A secure, privacy-first platform designed to bridge the gap between local shop owners and the blue-collar workforce.

## Key Features
- **Privacy-First:** OTP-gated profile access to protect worker information.
- **Flexible Hiring:** Supports daily, weekly, and monthly job posting models.
- **Efficient Matching:** Real-time job discovery for local workforce.
- **Secure Backend:** Built with Spring Boot and Spring Data JPA.

## Tech Stack
- **Backend:** Java 21, Spring Boot 3.x, MySQL, Spring Data JPA
- **Concurrency:** Java ExecutorService for background tasks
- **Security:** Secure API architecture with server-side validation

## Getting Started
1. Clone the repository.
2. Configure `application.properties` with your MySQL database credentials.
3. Run the application using `SahayakApplication.java`.
